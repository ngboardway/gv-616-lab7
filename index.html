Legal Statement

We hack for learning, not learn for hacking. You are not allowed to use the techniques gained from this course for any malicious purpose. Any malicious action is illegal and will be caught and prosecuted.

Educational Objectives

Network scanning is an important part of pen testing to find network vulnerabilities, including misconfigurations, unused open ports, old applications, out of date security packages, and etc. When we have collected such data, we will be able to harden the network and computing systems. This case study is designed for the following educational objectives:

  • Students will gain hands-on experience with ethical hacking; then they will be able to better harden a network and a system.

Lab Environment

You can use any VM and technology you are familiar with. However, you need to work on our isolated network. Most of the work can be conducted with the Kali VM.
